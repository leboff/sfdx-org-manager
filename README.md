# SFDX Org Manager

## Electron Desktop App for Managing SFDX Orgs

### Features

- List Authenticated Orgs (Scratch and Non-scratch orgs)
- Open Authenticated Orgs via App or Tray Icon
- Authenticate New Orgs

  - Edit Aliases
  - Change Defaults
